# Student Expense Tracker

A simple and responsive web application that helps students track their daily expenses, manage budgets, and understand spending habits.

## Project Overview

This project was created as a portfolio project to demonstrate front-end development skills using HTML, CSS, and JavaScript. Users can add expenses, choose categories, view total spending, and delete items when needed.

## Features

- Add expense name, amount, and category
- View total expenses instantly
- Delete expenses from the list
- Responsive design for mobile and desktop
- Simple and clean user interface
- Uses browser local storage to save data

## Technologies Used

- HTML
- CSS
- JavaScript
- Local Storage

1. Download or clone this repository.
2. Open `index.html` in your browser.
3. Start adding expenses.
