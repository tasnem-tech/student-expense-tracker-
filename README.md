# 🌍 Overview

**Student Expense Tracker** is a responsive web application designed to help students record daily expenses, monitor spending habits, and improve financial awareness.

The application provides a simple and intuitive interface for managing personal finances without requiring any backend or database.

---

# ✨ Features

| Feature | Description |
|----------|-------------|
| 💸 Add Expenses | Record expense name, amount and category |
| 📋 Expense List | View all recorded expenses |
| 🗑 Delete Expenses | Remove unwanted records |
| 📊 Total Spending | Automatically calculate total expenses |
| 💾 Local Storage | Save data directly in the browser |
| 📱 Responsive Design | Optimized for desktop, tablet and mobile |
